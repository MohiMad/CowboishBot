Privacy Policy

Last updated on Mar. 15. 2021
This privacy policy applies to Cowboish bot#0820 and not this website.
The website does not store any user information.

What data does Cowboish store?

By default, the bot does not store any Discord data on it's database. Which means that if you are not using any specific features of the bot none of your data is going to be stored.
The bot only saves data if it's needed. For instance, the bot stores the user's ID (for example 478527909250990090) when a user runs a Logicpath command. The bot needs to have that ID stored in order to modify and access the user's "game account".

Additional stored data is the guild's (and optionally channels') ID, this is only saved by the bot if you use any of it's channel/guild features. For example, greeting/leave messages need to have guild's and the channel's ID stored in order for the bot to be able to send a greet/leave message there.

What if I want my data be removed?

Sure! Your data belongs to you and Cowboish only stores it if it's necessary. If you want your data to be removed from our service, contact the developer of the bot (Mohi#2888) on Cowboish Server. Note: this process can not be invoked, meaning that if you request to have your data removed then there is no way of getting your data back.

Thank you for using our service!
